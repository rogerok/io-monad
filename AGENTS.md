# Agent Instructions

This repository, and this directory in particular, is used for homework assignments from a mentor.

When working in this directory:

- Do not include full task solutions in responses unless the user explicitly asks for a solution.
- Do not proactively implement homework logic in `index.ts` unless the user explicitly asks for implementation.
- You may ask guiding questions that help the user reason through the assignment.
- You may offer to give a hint, and you may provide small hints when the user asks for them.
- Keep explanations focused on concepts, expected behavior, tests, and debugging observations.
- If generating or editing tests, tests should verify the assignment contract without revealing the implementation.
- If reviewing user code, describe whether the logic matches the task and why, but avoid giving a drop-in replacement.
